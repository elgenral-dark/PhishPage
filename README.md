# 🎣 PhishPage

> **An educational cybersecurity project for learning phishing awareness, web security concepts, and defensive techniques in a safe and ethical environment.**

---

## 📖 Overview

**PhishPage** is an educational cybersecurity project designed to help learners understand phishing awareness, web security concepts, and defensive methodologies in a controlled environment. The project is intended for educational purposes, security research, and ethical learning only.
